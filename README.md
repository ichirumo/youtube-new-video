# youtube-new-video

初めてのdiscord bot 開発なので変なコードかもしれません。

## メイン機能

youtubeで新しい動画を投稿された場合、discordに送信

discord botのトークン暗号化

## サブ機能

・動画投稿された時にメンション

・Microsoft Translatorで翻訳

・botを動かしてるサーバーの状態

・discordの問題を確認

・ChatGPTを呼び出す(テストされていません)

・X(Twitter)に新しい投稿があると送信

・〇〇をプレイ中を設定可能
